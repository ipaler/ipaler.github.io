GitHub Pages + Hexo + GoDaddy + DNSPod = Blog

一直想折腾个自己的博客网站写点东西，之前想的解决方案是 GitHub Pages + Octopress，后来了解到 Hexo，可能因为后出的缘故，就选择它了。

Start

GitHub Pages

GitHub Pages 是 GitHub 为个人主页或项目主页提供的免费静态资源空间，可配合博客系统搭建自己的博客网站。

Hexo

Hexo 是基于 Node.js 的博客系统，只需简单的几个命令即可完成博客的生成与部署。Hexo 本地将博客生成静态页面，然后再将其部署至 GitHub。

Then

GoDaddy

GoDaddy 是一家域名服务商，支持支付宝结算。GitHub Pages 会提供一个二级域名供外界访问 http://username.github.io，如果你不满意这个默认域名，可以自行购买一个，然后与 GitHub 绑定使用。

DNSPod

DNSPod 是国内域名解析服务提供商，现被腾讯收购了，其免费服务够用。主要是 GoDaddy 自己的域名解析不稳定，而且速度也会受墙影响。

域名邮箱

有了自己的域名后，就可以使用域名邮箱服务，自定义邮箱地址。推荐腾讯企业邮箱。

End

了解以上信息之后，就可以按照以上介绍的顺序一个个去弄了。每项服务都有新手向导，链接和关键词也都给出来了。
